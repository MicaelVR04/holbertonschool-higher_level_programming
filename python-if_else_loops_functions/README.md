# Holberton School - Higher Level Programming
Simple exercises for practicing Python control flow and functions.

## Description
This directory contains small Python scripts and tasks that focus on:
- if, elif, else statements
- for and while loops
- defining and using functions

## Requirements
- Python 3.x

## Usage
Run examples with:
```
python3 <script>.py
```

## Structure
- Each task is a standalone `.py` file named according to the assignment.

## Notes
Keep solutions simple, readable, and follow PEP 8 where practical.
